# Carte Blanche

Flashcards mobile pour apprendre les départements français — numéro ↔ nom.

## Stack

- [Expo](https://expo.dev) + React Native
- [React Native Paper](https://reactnativepaper.com) (Material Design 3)
- TypeScript

## Fonctionnalités

- Retournement de carte animé
- Mode normal (numéro → nom) et inversé (nom → numéro)
- Tirage aléatoire
- Recherche par numéro ou nom
- Thème clair / sombre

## Lancer le projet

```bash
npm install
npm start
```
